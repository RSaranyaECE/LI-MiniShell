# LI-MiniShell

Implemented Features:

1. PS1=NEWPROMPT

2. Executable commands and executable files

3. Special variables :

        echo $?

        echo $$

        echo $SHELL

4. Signal handlers :  

        ctrl + c

        ctrl + z

5. Built -in commands:

        exit

        cd

        pwd

6. Background and foreground process:

        fg

        bg

        jobs
        
7. pipe functionality
        
        ls | gerp a | wc
